정오표 — Errata

독자가 발견하신 오류, 또는 저자가 추후 수정한 사항을 한 자리에 모았습니다. 모든 수정 사항은 다음 v1.x 빌드에 즉시 반영되며, 기존 구매자는 무료 업데이트를 받으실 수 있습니다.

v1.0 오류 목록

페이지 분류 잘못된 내용 수정 내용 반영
아직 보고된 오류가 없습니다. 독자분께서 처음으로 발견하신 오류는 다음 빌드에 반영하고 이 페이지에 기록됩니다.

오류 신고

사실 오류 · 오탈자 · 코드 버그 · 출처 표기 누락을 발견하시면 다음 정보와 함께 알려 주십시오:

  • 페이지 번호 와 해당 단락의 첫 5자 정도
  • 잘못된 내용 과 제안하시는 수정안
  • 가능하시다면 출처(공식 문서 / PEP / CPython 라인 등)
