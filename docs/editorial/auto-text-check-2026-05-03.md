## 분류별 결과

### space_before_punct (n≈580)
패턴: ` ,` 또는 ` :` — 본문에서 단어와 구두점 사이 공백.
- 거의 전부 HTML 마크업 (`<br>\n      :`, 표 셀 줄바꿈) 잔재
- 실제 본문 오타 아님 → **무시**

### duplicate_word (n≈82)
패턴: `같은단어 같은단어`.
- 표 셀에서 같은 값 반복 (False False, True True)
- 코드 결과 셀 (PyFloatObject PyFloatObject)
- 실제 의심 케이스:
  - week04: `hahaha hahaha` — 본문 강조일 가능성, 수동 확인 필요
  - week14: `이 이`, `Optional Optional` — 조사 중복일 수 있음
  - week12: `이 이`, `예외 예외` — 동일

### mixed_quotes (n≈506)
패턴: `"..."` 한 줄에 따옴표 혼용.
- 대부분 코드 속성 (`"data-theme"`) — 정상
- 본문 한국어 인용에 영문 직선따옴표(`"`) 사용 — 한국어 표준은 곡선따옴표(`"..."`)
- 일관성 통일 작업은 가독성에 큰 영향 X → P3로 분류

### 의도된 에러 (false positive)
- 책 본문에서 일부러 보여주는 잘못된 코드 / 잘못된 표기
- 학습 목적 → 수정 안 함
